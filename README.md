# Lacaille-Therapy
Este projeto é o website fictício de um consultório de psicologia, a Lacaille Therapy. Ele foi desenvolvido com o objetivo principal de aprofundar e praticar habilidades de desenvolvimento web simulando um ambiente real de um site profissional.
Embora o consultório e a psicóloga sejam fictícios para fins de estudo, o projeto busca seguir as melhores práticas de design e usabilidade para um site na área da saúde mental.

Tecnologias Utilizadas

HTML5: Estrutura semântica do conteúdo.

CSS3: Estilização, layout responsivo e animações.

JavaScript (Vanilla JS): Interatividade do lado do cliente (como formulários, navegação e sliders).
